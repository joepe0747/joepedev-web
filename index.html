Sobre mí

Hecho en Argentina,
para el mundo

Soy José María, desarrollador Android independiente con base en Buenos Aires, Argentina. Me especializo en construir aplicaciones móviles reales, con arquitectura MVVM, Firebase y Jetpack Compose. No trabajo para una empresa: cada app que publico la imagino, la diseño y la lanzo de forma independiente. Eso me da la libertad de iterar rápido, escuchar a los usuarios y tomar decisiones de producto sin burocracia.

Actualmente mantengo tres proyectos activos: Chat Caliente, una comunidad de chat social con verificación de identidad por DNI y moderación con IA; DailyBoost Widget, un widget de frases motivacionales para la pantalla de inicio de Android; y NeuroKids Ayuda, una app educativa adaptativa para niños con neurodiversidad. Todas publicadas en Google Play con usuarios reales en Argentina y Latinoamérica.

Mi stack principal incluye Kotlin, Jetpack Compose, Firebase Realtime Database, Firebase Storage, Firebase Auth, ML Kit para visión por computadora, CameraX para captura de imagen y video, y Cloud Functions para la lógica de backend. También trabajo con GitHub Actions para CI/CD y despliego sitios estáticos con GitHub Pages y Cloudflare.
